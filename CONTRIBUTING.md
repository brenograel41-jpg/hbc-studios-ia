# Contribuindo para GHS STUDIOS IA

Adoramos contribuições! Este documento fornece diretrizes.

## 🚀 Como Começar

### 1. Fork e Clone
```bash
git clone https://github.com/seu-usuario/hbc-studios-ia.git
cd hbc-studios-ia
```

### 2. Crie uma Branch
```bash
git checkout -b feature/sua-feature
```

### 3. Configure o Ambiente
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 📋 Diretrizes

- Siga [PEP 8](https://pep8.org/)
- Use type hints
- Escreva docstrings
- Commits com prefixo: `feat:`, `fix:`, `docs:`

## 🧪 Testes

```bash
pytest
pytest --cov=src/
flake8 src/
```
